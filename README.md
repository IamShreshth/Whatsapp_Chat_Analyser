# Whatsapp_Chat_Analyser
WhatsApp Chat Analyzer
Description:
A C program that analyzes a WhatsApp chat export (chat.txt) and shows:
- Total messages sent by each user
- Average message length per user
- Most active hour of the day
- Message count per hour
Output Meaning:
Messages - Total messages sent by each user
Avg length - Average number of characters per message
Most Active Hour - Hour when most messages were sent
Messages per Hour - Number of messages in each hour
